# 🧠 Transtypage et `instanceof` en Java

## 🔹 Qu’est-ce que le transtypage (casting) ?

Le **transtypage** consiste à convertir une référence d’un type vers un autre type dans une hiérarchie de classes.

### Types de transtypage :
- **Upcasting (implicite)** : sous-classe → superclasse  
- **Downcasting (explicite)** : superclasse → sous-classe (⚠️ nécessite vérification)

```java
Animal a = new Grenouille(); // Upcasting automatique

Grenouille g = (Grenouille) a; // Downcasting (risqué si a n'est pas une Grenouille)
```

---

## 🔹 Pourquoi utiliser `instanceof` ?

L’opérateur `instanceof` permet de **vérifier le type réel d’un objet** avant de faire un transtypage.

👉 Évite les erreurs de type (`ClassCastException`)

```java
if (a instanceof Grenouille) {
    Grenouille g = (Grenouille) a;
}
```

---

## Exemple complet de transtypage : Animal, Grenouille, Baleine

### Classe de base

```java
abstract class Animal {
    protected String nom;

    public Animal(String nom) {
        this.nom = nom;
    }

    public String getNom() {
        return nom;
    }
}
```

---

### Sous-classes

```java
class Grenouille extends Animal {
    public Grenouille(String nom) {
        super(nom);
    }

    public boolean saute() {
        return true;
    }
}

class Baleine extends Animal {
    public Baleine(String nom) {
        super(nom);
    }

    public boolean nage() {
        return true;
    }
}
```

---

## Utilisation avec une liste d’animaux

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {

        List<Animal> animaux = new ArrayList<>();

        animaux.add(new Grenouille("Kermit"));
        animaux.add(new Baleine("Willy"));
        animaux.add(new Grenouille("Froppy"));
        animaux.add(new Baleine("Moby"));
        animaux.add(new Grenouille("Pepe"));

        System.out.println("Nom\t\tType\t\tSaute\t\tNage");
        System.out.println("------------------------------------------------");

        for (Animal a : animaux) {

            String type = "";
            String saute = "-";
            String nage = "-";

            if (a instanceof Grenouille) {
                type = "Grenouille";
                Grenouille g = (Grenouille) a;
                saute = g.saute() ? "Oui" : "Non";
            } 
            else if (a instanceof Baleine) {
                type = "Baleine";
                Baleine b = (Baleine) a;
                nage = b.nage() ? "Oui" : "Non";
            }

            System.out.println(a.getNom() + "\t\t" + type + "\t\t" + saute + "\t\t" + nage);
        }
    }
}
```
