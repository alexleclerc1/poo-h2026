# Transtypage et `instanceof` en Java

## 1. Qu'est-ce que le transtypage (casting)?

> **Définition** : Transtypage  
Le **transtypage** (ou conversion de type) consiste à convertir une référence d'un type vers un autre type dans une hiérarchie de classes.

### Syntaxe

On inscrit le type dans lequel on veut convertir entre parenthèses `( )` devant la variable référence à transtyper.

```java
int entier = 10;
double reel = (double) entier;
System.out.println(reel);       // 10.0
```

L'exemple ci-dessus est appliqué à des types primitifs, `int` et `double`. En pratique, nous allons utiliser le transtypage sur des **objets** de classes ayant des relations d'héritage entre elles.

### Types de transtypage

- **Upcasting** (implicite) : sous-classe → superclasse. **Sécuritaire**, car une sous-classe **EST UNE** superclasse.
- **Downcasting** (explicite) : superclasse → sous-classe. **Dangereux** (⚠️ nécessite vérification), car une superclasse n'est pas nécessairement une sous-classe.

```java
// Exemples de transtypages
Animal a = new Grenouille();    // Upcasting : une Grenouille EST UN Animal
Grenouille g = (Grenouille) a;  // Downcasting : (risqué si a n'est pas une Grenouille)
```

## 2. Opérateur `instanceof`

L'opérateur `instanceof` permet de **vérifier le type réel d'un objet** avant de faire un transtypage. Il retourne une valeur booléenne.

> **REMARQUE** :  
Pour une classe donnée, disons `MaClasse`, l'opérateur `instanceof` retourne `true` pour la classe elle-même **mais aussi pour toutes les classes parentes** de `MaClasse`, et ce jusqu'à la classe `Object`.

👉 Cet opérateur sert à éviter les erreurs de conversion de type (`ClassCastException`) :

```java
a instanceof Grenouille;    // true
a instanceof Animal;        // true
a instanceof Object;        // true
a instanceof Baleine;       // false

if (a instanceof Grenouille) {
    // Il est sécuritaire de transtyper a en un objet Grenouille ici
    Grenouille g = (Grenouille) a;
}
```

## 3. Exemple complet de transtypage : `Animal`, `Grenouille`, `Baleine`

### Classe de base `Animal` (abstraite)

```java
abstract class Animal {
    protected String nom;

    public Animal(String nom) {
        this.nom = nom;
    }

    public String getNom() {
        return nom;
    }
}
```

### Classes dérivées `Grenouille` et `Baleine`

```java
class Grenouille extends Animal {
    public Grenouille(String nom) {
        super(nom);
    }

    public boolean saute() {
        return true;
    }
}

class Baleine extends Animal {
    public Baleine(String nom) {
        super(nom);
    }

    public boolean nage() {
        return true;
    }
}
```

## 4. Utilisation avec une liste d'animaux

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {

        List<Animal> animaux = new ArrayList<>();
        final String FORMAT = "%-12s%-15s%-8s%-8s";

        animaux.add(new Grenouille("Kermit"));
        animaux.add(new Baleine("Willy"));
        animaux.add(new Grenouille("Froppy"));
        animaux.add(new Baleine("Moby"));
        animaux.add(new Grenouille("Pepe"));

        System.out.println(String.format(FORMAT, "Nom", "Type", "Saute", "Nage"));
        System.out.println("---------------------------------------");

        for (Animal a : animaux) {

            String type = "";
            String saute = "-";
            String nage = "-";

            if (a instanceof Grenouille) {
                type = "Grenouille";
                Grenouille g = (Grenouille) a;
                saute = g.saute() ? "Oui" : "Non";
            } 
            else if (a instanceof Baleine) {
                type = "Baleine";
                Baleine b = (Baleine) a;
                nage = b.nage() ? "Oui" : "Non";
            }

            System.out.println(String.format(FORMAT, a.getNom(), type, saute, nage));
        }
    }
}
```

### Sortie du programme

```text

---------------------------------------
Kermit      Grenouille     Oui     -       
Willy       Baleine        -       Oui     
Froppy      Grenouille     Oui     -       
Moby        Baleine        -       Oui     
Pepe        Grenouille     Oui     -
```
